# 🔍 HH Parser

Парсер вакансий с hh.ru с удобным веб-интерфейсом. Собирает вакансии по поисковому запросу и сохраняет в Excel-файл.

![Python](https://img.shields.io/badge/Python-3.11-blue) ![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey)

## Что умеет

- Поиск вакансий по любому запросу (`"CRM-маркетолог" OR "retention manager"`)
- Фильтр по городу или удалённая работа
- Сохранение в Excel с форматированием, гиперссылками и фильтрами
- Лимит на количество вакансий
- Кнопка остановки в любой момент

## Скачать готовую сборку (Windows)

1. Открой вкладку [Actions](../../actions)
2. Нажми на последний успешный запуск (зелёная галочка ✅)
3. Листай вниз до раздела **Artifacts**
4. Скачай **HH_Parser_Windows** (ZIP ~376 MB)
5. Распакуй ZIP
6. Запусти `HH_Parser.exe`

> ⚠️ **Windows Defender** может показать предупреждение при первом запуске.  
> Нажми **«Подробнее»** → **«Выполнить в любом случае»** — это стандартное поведение для неподписанных приложений.

## Использование

1. Введи поисковый запрос
2. Выбери город
3. Укажи имя файла и максимум вакансий
4. Нажми **▶ Запустить**
5. При первом запуске войди в hh.ru в открывшемся браузере — сессия сохранится

Файл сохраняется в папку `Загрузки` (`Downloads`).

## Запуск из исходного кода

```bash
pip install playwright openpyxl gradio
playwright install chromium
python hh_gradio.py
```

## Стек

- [Playwright](https://playwright.dev/python/) — управление браузером
- [Gradio](https://gradio.app/) — веб-интерфейс
- [openpyxl](https://openpyxl.readthedocs.io/) — генерация Excel
